## Scalability Plan

### Current Capacity (Single Server)
- **Requests/sec**: ~50
- **Concurrent users**: ~200
- **DB connections**: 20 pool size
- **Bottleneck**: PostgreSQL queries

### Phase 1: Redis Cache
- **Expected improvement**: 5x faster session loads
- **New capacity**: ~100 req/sec, 500 concurrent users
- **Cost**: +$11/month

### Phase 2: Horizontal Scaling
- **Setup**: Load balancer + 3 app servers
- **Expected capacity**: ~300 req/sec, 1500 concurrent users
- **Cost**: +$50/month

### Phase 3: Database Read Replicas
- **Setup**: 1 write master + 2 read replicas
- **Expected capacity**: ~500 req/sec, 3000 concurrent users
- **Cost**: +$20/month
